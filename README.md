# fuzzy-file-search

Quickly jump to files in vim

## Installing

Add the following to your .vimrc

```Vim
set path+=**
set wildmenu
```

## Usage

* `:find` to search for and open file in current directory
* `:b` to search within buffers
* `:ls` to list open buffers
